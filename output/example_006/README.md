# Example 006 - SourceFaces + Online Face Mix

This pack contains 10 new face images generated by combining:
- Local `SourceFaces` images
- 10 online face references discovered and downloaded from Wikimedia Commons

## Requirements Met
- Online references checked and grabbed: `10`
- New generated images: `10`
- Output dimensions: `384x615` (same as SourceFaces)
