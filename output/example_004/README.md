# Example 004 - NASA-Inspired Sci Faces

This folder contains generated sci-fi portrait graphics inspired by NASA-style astronaut visuals.

## Included Variants
- `Assets/Resources/SciFaces/nasa_sci_man_01.png`
- `Assets/Resources/SciFaces/nasa_sci_man_02.png`
- `Assets/Resources/SciFaces/nasa_sci_man_03.png`
- `Assets/Resources/SciFaces/nasa_sci_woman_01.png`
- `Assets/Resources/SciFaces/nasa_sci_woman_02.png`
- `Assets/Resources/SciFaces/nasa_sci_woman_03.png`

All portraits are 512x512 PNG images for easy use in Unity UI, cards, or character select screens.

## Unity Usage Example
```csharp
Sprite portrait = Resources.Load<Sprite>("SciFaces/nasa_sci_woman_02");
```

Design note:
- These are NASA-inspired in color and astronaut theme.
- No NASA logos were used.
